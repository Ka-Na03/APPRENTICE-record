# エンティティを定義できる

## 1. エンティティ

- ユーザー情報
  - ユーザーID
  - ユーザー名
  - ユーザーパスワード(暗号化)
  - 性別
  - 年齢
  - 生年月日
  - 登録日時
- 商品
  - 商品ID
  - 商品名
  - 価格
  - メーカーID
  - カテゴリーIDもしくはカテゴリー名
  - 在庫数
- 購入履歴
  - 購入ID
  - 購入したユーザーID
  - 購入された商品ID
  - 購入された商品価格
  - 購入日時
- メーカー
  - メーカーID
  - メーカ名
  - メーカーの詳細情報(連絡先や所在地など)
- カテゴリー
  - カテゴリーID
  - カテゴリー名
